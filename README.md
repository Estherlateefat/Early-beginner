Form Student Information
A simple HTML form for collecting student information.
Features
Collects full name, email, password, age, gender, profile pic (JPG), hobbies, country, and bio
Basic form validation (required fields, email format, age range)
Submit and Reset buttons
Usage
Open `form.html` in a browser to view the form
Fill in details and submit
